# AMMDS 离线版说明文档

## 项目概述

AMMDS（Adult Movie MetaData Scraper）是一款私人专属数据管理平台，专注于帮助用户轻松管理和整理个人影视库。通过智能刮削元数据、演员识别、多源数据匹配等功能，打造专属的家庭影院体验。

- 版本：1.6.31
- 架构：x86_64
- 开发者：萌森工作室
- 最低系统版本要求：0.9.0

## 功能特性

- 智能刮削元数据
- 演员识别
- 多源数据匹配
- 个人影视库管理
- 家庭影院体验优化

## 安装配置

### 安装步骤

在安装过程中，您需要完成以下配置：

1. **管理员密码设置**
   - 默认管理员用户名：ammds
   - 设置管理员密码（至少8位）

2. **应用配置**
   - 设置应用端口（默认：9523）
   - 阅读并同意服务条款和用户隐私保护协议

### 访问信息

- 默认端口：9523
- 默认用户名：ammds
- 密码：安装时设置的管理员密码

## 启动和停止服务

### 启动服务

```bash
./cmd/main start
```

### 停止服务

```bash
./cmd/main stop
```

### 查看服务状态

```bash
./cmd/main status
```

## 卸载说明

在卸载应用时，您可以选择如何处理应用数据：

1. **保留数据（推荐）** - 将来重新安装时可恢复
2. **删除所有数据** - 此操作不可恢复，请谨慎选择
